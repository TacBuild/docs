TAC EVM is a standard EVM chain based on CosmosSDK + Ethermint. These are the chain parameters for TAC:

- **Turin (Testnet):**

  - RPC endpoint: https://turin.rpc.tac.build
  - Blockexplorer: https://turin.explorer.tac.build
  - Faucet (for gas fees): https://turin.faucet.tac.build
  - ChainId: 2390
  - FeeSchema: EIP-1559
  - BlockGasLimit: 20M
  - Blocktime: 3 seconds

- **Saint Petersburg (Testnet):**
  - RPC endpoint: https://spb.rpc.tac.build
  - Blockexplorer: https://spb.explorer.tac.build
  - Faucet (for gas fees): https://spb.faucet.tac.build
  - ChainId: 2391
  - FeeSchema: EIP-1559
  - Blocktime: 2 seconds

<Warning>
  TAC Turin is going to be deprecated soon. Please use Saint Petersburg Testnet
  for your nodes and applications.
</Warning>
